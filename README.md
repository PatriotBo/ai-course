# AI Course Workspace

这是为后端工程师转 AI 应用 / Agent 开发搭建的长期课程工作区。

---

## 目录结构

```text
ai-course/
├── README.md                  # 本说明
├── COURSE_PLAN.md             # 12 周课程总纲
├── PROGRESS.md                # 学习进度 Ledger
├── GLOSSARY.md                # AI 术语表
├── PROJECTS.md                # 项目档案
├── lessons/                   # 每节课讲义
├── assignments/               # 每节课作业
├── reviews/                   # 作业批改与阶段复盘
├── code/                      # 课程配套代码
└── references/                # 参考资料
```

---

## 使用方式

### 开始课程

```text
开始 Lesson 0
```

或：

```text
开始今天的 AI 课程
```

### 提交作业

```text
这是 Lesson X 的作业，请批改：
...
```

### 查看进度

阅读：`PROGRESS.md`

---

## 授课规则

每节课固定流程：

1. 回顾上节课；
2. 检查作业；
3. 输出今日备课单；
4. 正式授课；
5. 代码实战；
6. 课堂练习；
7. 布置作业；
8. 更新进度。
